# Support

If you need help with Mahina OS, want to ask questions about the architecture, or simply want to chat with the community, you are in the right place!

## Where to get help

1. **The Divine Collection of Knowledge about Luna (DCKL)**
   Before asking a question, please check the `docs/DCKL/` directory. Almost every design decision, architecture flow, and technical requirement is exhaustively documented there. Start with `Volume I - Foundation`.

2. **GitHub Discussions**
   For general questions, brainstorming, or asking "how do I do X?", please use [GitHub Discussions](https://github.com/HardikBhaskar2010/MahinaOS/discussions). 

3. **GitHub Issues**
   If you have found a reproducible bug or want to propose a new feature (that aligns with the Roadmap), please open an [Issue](https://github.com/HardikBhaskar2010/MahinaOS/issues). 
   *Note: For security vulnerabilities, please refer to `SECURITY.md`.*

4. **Discord (Coming Soon)**
   As the community grows, we plan to launch an official Discord server for real-time collaboration. Stay tuned!
